# 需求

基于Garmin手表的数据来将跑步数据展示在Web页面，基础效果就只需要能够本地起服务，把现有的数据展示出来即可

# 技术栈

需要和AI进行讨论

基于 Cline + Deepseek + React + TypeScript + Tailwind CSS 构建

# 数据来源

Garmin手表的数据会以fit存储，已经有一些现有工具可以参考来解析Garmin数据，比如https://github.com/sivaprakasaman/garmin_wrapped 基于R语言解析fit文件。需要注意导出的不只有跑步数据，还有力量训练、徒步等冗余数据，需要进一步过滤。

# 优先级

首先可以先把Garmin数据解析好，然后先通过图表形式展示获取到的数据

之后再考虑数据存储和网页制作
